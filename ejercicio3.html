Calculadora de envío con fragmentos DOM

Definiciones

Una encomienda consta de uno o más bultos. El peso de cada bulto se define como el mayor entre el peso físico y el peso volumétrico, calculado como ancho × largo × alto dividido por un factor volumétrico.

La entrada de datos se realiza con formularios HTML y la salida de datos se muestra en tablas HTML creadas con fragmentos de DOM.
